# RADIOMASTER 通过usb获取键位值
# 订阅 joy 话题即可
# 键位介绍
- axes[0]:右摇杆左右 左负右正
- axes[1]:右摇杆上下 下负上正
- axes[2]:左摇杆上下 下负上正
- axes[3]:左摇杆左右 左负右正
- axes[4]:左拨轮 按下为正 0.99 
- axes[5]:左按键 - 0 +
- axes[6]:右按键 - 0 +
- axes[7]:右拨轮 按下为正 0.99 
- axes[8]:左下案件 按下为正
- axes[9]:右滑轮 左负右正

正常编译使用即可
